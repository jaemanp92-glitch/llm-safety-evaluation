# LLM Safety Evaluation Project

## Overview
This project evaluates Large Language Model (LLM) safety responses across multiple risk categories including:

- Self-harm
- Privacy
- Fraud
- Cybersecurity
- Jailbreak attempts
- Harassment

The dataset was manually created and analyzed using Python and Google Colab.

---

## Project Files

- `llm_safety_dataset_v1.csv`
  - Structured evaluation dataset

- `Untitled1.ipynb`
  - Python analysis notebook

- `safety_score_chart.png`
  - Visualization of average safety scores by category
